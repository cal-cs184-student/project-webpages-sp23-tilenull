Part 3: Direct Illumination

We loop over num_samples samples and for each sample we sample a ray direction, wj, from hemisphereSample. We use this to generate a ray from hit_p in the direction wj. Because wj is sampled in object space, we must convert it to world space by multiplying by the object to world matrix, o2w and normalize. We set the min_t of this new ray to EPS_F, a small constant that prevents numerical precision issues where the ray intersects the surface it starts from. In order to check if the new ray adds direct illumination to the light source we use bvh->intersect. If it does intersect, we can add this sample to our running total. We multiply fr, the BSDF by Li, the incoming light calculated from get_emission() of the intersection and then multiply by cos_theta(wj) and then normalize by dividing by the probability of this sample, p(wj), which is the pdf of the hemisphere, 1/2pi. Finally, we normalize the total by dividing by the number of samples.
